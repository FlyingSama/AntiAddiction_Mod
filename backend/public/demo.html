API 端点参考
端点说明
GET /api/rules获取游戏日规则和默认时段（Mod 客户端调用）
POST /api/verify实名认证并返回签名 credential
POST /api/report携带 credential 上报辅助审计事件
POST /api/report/batch携带 credential 批量上报辅助审计事件
GET /api/admin/*管理后台 API（需管理员登录）
